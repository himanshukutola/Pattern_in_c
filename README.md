#PATTERN IN C
This project contains C programs that generate various patterns using loops and logic. The purpose is to practice and demonstrate different pattern printing techniques in C.

FEATURES

Various pattern types, such as:
Hollow Square.
Right Triangle.
Inverted Triangle.
Diamond Patterns.


USAGE

Compile the program using a C compiler 
Run the executable and choose the pattern type via the menu.

FUTURE IMPROVEMENTS

Add dynamic pattern sizes based on user input.
Implement additional patterns (e.g., pyramids, circles).
